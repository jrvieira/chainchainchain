# chainchainchain
Liberal, unobtrusive pseudo-inheritance JavaScript micro library for OOP.

##Installation

npm

	$ npm install chainchainchain

bower

	$ bower install chainchainchain

##Usage

```javascript
const chain = require('chainchainchain');
```
