Documentation


This non-linear pseudo-inheritance micro library allows for independent chain objects to be created. You can return properties and methods of chained objects directly through your chain objects and manipulate and evaluate your chain objects via the methods described in this document. You can import this module any way you like but for the purpose of this document let's assume the following:

const chain = require('chainchainchain');

( obj [, obj , ...] ) Create a chain object.
obj object
Objects from which to inherit.
returns { Chain object }
Mind that the rightmost objects always become the oldest in chain as this method adds to the top of the chain. If you want to add to the bottom of the chain, use .
set.owncontext property
Bind methods to their corresponding object's context instead of the origin object. Defaults to false.

The origin object is the youngest one, at the bottom of the chain.
set.uniqueness
This setting has been deprecated.
set.allowloops property
Allow for inheritance loops when chaining chain objects. Defaults to false.
set.setchained property
Set values on the youngest chained object with the existing property instead of the origin. Defaults to false.
add ( ch , obj ) Add objects to the top of the chain.
ch { Chain object }
Target chain.
obj object/array
Object or array of objects to add to the top of the chain.
returns array
Mind that the rightmost objects always become the eldest in chain as this method adds to the top of the chain. If you want to add to the bottom of the chain, use .
Returns an array of chained objects.
pre ( ch , obj ) Add objects to the bottom of the chain.
ch { Chain object }
Target chain.
obj object/array
Object or array of objects to add to the bottom of the chain.
returns array
Mind that the leftmost objects always become the youngest in chain as this method adds to the bottom of the chain. If you want to add to the top of the chain, use .
Returns an array of chained objects.
rep ( ch , x, o ) Replace object in chain.
ch { Chain object }
Target chain.
x object
Object to replace from chain.
o object
Object to replace x with.
returns array
Mind that this method replaces all instances of x in the chain.
Returns an array of chained objects.
rem ( ch , x ) Remove objects from the chain.
ch { Chain object }
Target chain.
obj object/array
Objects to remove.
returns object
Mind that this method removes all instances of x in the chain.
Returns an array of chained objects.
pop ( ch ) Remove the top object from the chain.
ch { Chain object }
Target chain.
returns object
Returns the removed object, like Array.prototype.pop.
shift ( ch ) Remove the bottom object from the chain.
ch { Chain object }
Target chain.
returns object
Returns the removed object, like Array.prototype.shift.
rol ( ch , n ) Rotate the entire chain.
ch { Chain object }
Target chain.
n number
Number of rotations. Negative numbers invert the direction of the rotation.
returns array
If n > 0 the bottom object moves to the top of the chain.
If n < 0 the top object moves to the bottom of the chain.
Returns an array of chained objects.
is ( z ) Check for a chain object.
z any
Target.
returns boolean
Returns true if z is a chain object.
origin ( ch ) Get origin object.
ch { Chain object }
Target chain.
returns object
The origin object is the youngest one, at the bottom of the chain.
arr ( ch ) Get all objects in chain.
ch { Chain object }
Target chain.
returns array
raw ( ch , prop [, callback] ) Get each chained object's selected property.
ch { Chain object }
Target chain.
prop string
Name of the property or method you want to return.
callback function
Callback function that takes two arguments: an array of returned properties and an array of their corresponding chained objects.
returns array/any
Beware of recursion when creating trees by chaining chain objects.
Returned methods are binded to the origin object's context by default. See .

The origin object is the youngest one, at the bottom of the chain.
